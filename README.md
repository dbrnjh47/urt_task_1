# Тестовое задание компании Media Army задача #1

Необходимо выполнить верстку по дизайн-макету с учетом следующих условий:
при верстке страницы не должны использоваться какие-либо фреймворки, библиотеки, сборщики и прочее;
результатом работы должны быть 3 файла: html, css, js, можно собрать их в архив ZIP и приложить как результат;
адаптивность с помощью @media запросов на свой вкус, но чтобы на мобильных устройствах ничего не уезжало;
никаких float'ов и таблиц, только flex или grid;
никаких изображений;
чистый, нативный js для открытия окна.
